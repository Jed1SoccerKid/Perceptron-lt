# Perceptron-xo

# X vs O Perceptron Classifier

A perceptron classifier that predicts whether a drawn symbol is an X or O.

## How it works
- Draw on the 5x5 grid by clicking the cells
- Click Classify to get a prediction
- Uses a perceptron trained on 100 samples (50 X, 50 O)

## Files
- index.html - the web app
- xo_data.csv - the training dataset (100 annotated samples)

## How to run
Open the GitHub Pages link in any browser.

## Built with
- PyScript
- Python (perceptron trained in Google Colab)
